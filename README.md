# 🛒 E-Commerce Platform

A robust, full-stack e-commerce web application designed to provide a seamless online shopping experience. This platform features user authentication, product browsing, a dynamic shopping cart, and secure order processing.

## 🚀 Tech Stack

### Frontend
* **React.js:** UI component building and state management (using React Hooks).
* **HTML5/CSS3/Tailwind4.3/Framer0Motion:** Structure and styling.

### Backend
* **Java & Spring Boot:** RESTful API development and core business logic.
* **Spring Security:** Authentication and authorization.

### Database
* **MongoDB:** NoSQL database for flexible product catalogs and unstructured data.


### Tools & Version Control
* **Git & GitHub:** Source code management and version control.
* **Maven:** Dependency management for the Java backend.
* **npm/yarn:** Package managers for the React frontend.

## ✨ Key Features

* **User Authentication:** Registration, login, and secure session management.
* **Product Catalog:** Browse products by category, filter, and search functionality.
* **Shopping Cart:** Add, remove, and update quantities of items in the cart.
* **Order Management:** Secure checkout process and user order history tracking.
* **Admin Dashboard:** Interface for administrators to manage products, users, and view sales.
